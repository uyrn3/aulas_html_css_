ID e CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um numero de cpf, cada elementos pode ter um valor unico, ou seja ninguem pode ter um cpf(id) igual.

CLASS

O class funciona de maneira oposta ao ID. Diferemte do ID em que você só pode ter um valor, no class você pode ter vários valores. Também diferente do id, em que não se pode repetir um valor, no class você pode ultizar o o mesmo valorpara varios elementos
