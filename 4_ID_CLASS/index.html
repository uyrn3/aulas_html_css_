ID de CLASS

ID

O ID funciona como um CPF. Assim como só posso ter número de CPF cada elemento só pode ter um valor unico ou seja ninguem pode ter um CPF(ID) igual. Ele é unico então não pode repitir.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você so pode ter um valor, no CLASS você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode repetir um valor, no CLASS você pode utilizar o mesmo valor em varios elementos.
